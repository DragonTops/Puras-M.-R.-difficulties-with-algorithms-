# Лабораторная работа 5: Хеш-функции и хеш-таблицы

## Описание проекта
Исследование хеш-функций и методов разрешения коллизий в хеш-таблицах. Реализация различных стратегий хеширования и сравнительный анализ их эффективности.

## Цели работы
- Изучить принципы работы хеш-функций и хеш-таблиц
- Освоить методы разрешения коллизий
- Реализовать хеш-таблицы с разными стратегиями разрешения коллизий
- Провести сравнительный анализ эффективности методов
