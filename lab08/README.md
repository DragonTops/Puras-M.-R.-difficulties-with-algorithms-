# Лабораторная работа 8: Жадные алгоритмы

## Описание проекта
Исследование и реализация жадных алгоритмов для решения классических задач. Анализ корректности и эффективности жадного подхода.

## Цели работы
- Изучить метод проектирования алгоритмов "жадный алгоритм"
- Освоить принцип принятия локально оптимальных решений
- Реализовать классические жадные алгоритмы
- Проанализировать корректность и эффективность жадного подхода


## Реализованные алгоритмы

### Основные алгоритмы
- **Задача о выборе заявок** - выбор максимального количества непересекающихся интервалов
- **Непрерывный рюкзак** - максимизация стоимости с возможностью брать дробные части
- **Алгоритм Хаффмана** - оптимальное префиксное кодирование
- **Задача о монетах** - минимальное количество монет для сдачи

### Дополнительные алгоритмы
- **Алгоритм Краскала** - минимальное остовное дерево
