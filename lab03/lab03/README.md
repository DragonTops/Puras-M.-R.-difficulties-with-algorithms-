# Лабораторная работа 3: Рекурсия

## Описание проекта
Исследование и реализация рекурсивных алгоритмов. Сравнительный анализ производительности рекурсивных и итеративных реализаций базовых алгоритмов. Решение практических задач с использованием рекурсии.

## Цели работы
- Изучить понятие рекурсии и особенности рекурсивных алгоритмов
- Реализовать рекурсивные версии базовых алгоритмов
- Сравнить эффективность рекурсивных и итеративных реализаций
- Решить практические задачи с использованием рекурсии

## Результаты

### Производительность операций
| Алгоритм | Рекурсивная версия | Итеративная версия |
|----------|-------------------|-------------------|
| Факториал | O(n) | O(n) |
| Числа Фибоначчи | O(2^n) | O(n) |
| Бинарный поиск | O(log n) | O(log n) |
| НОД | O(log min(a,b)) | O(log min(a,b)) |

### Ключевые выводы
1. Рекурсивные алгоритмы могут быть менее эффективными из-за накладных расходов на вызовы функций
2. Для чисел Фибоначчи итеративная версия значительно быстрее рекурсивной
3. Некоторые алгоритмы (бинарный поиск, НОД) имеют схожую производительность в обеих версиях
4. Рекурсия удобна для решения задач с древовидной структурой (обход деревьев, Ханойские башни)
