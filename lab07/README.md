# Лабораторная работа 7: Кучи (Heaps)

## Описание проекта
Исследование и реализация структуры данных "куча", алгоритма сортировки кучей (Heapsort) и приоритетной очереди. Анализ производительности операций и сравнение с другими алгоритмами.

- Изучить свойства и особенности структуры данных "куча"

- Реализовать приоритетную очередь на основе кучи
- Провести сравнительный анализ производительности

## Основные функции

### Реализованные структуры данных
- **MinHeap** и **MaxHeap** с операциями:
  - Вставка элемента: O(log n)
  - Извлечение корня: O(log n)
  - Построение кучи из массива: O(n)

### Алгоритмы
- **Heapsort**: стабильная сортировка за O(n log n)
- **In-place Heapsort**: сортировка без дополнительной памяти

### Приоритетная очередь
- **PriorityQueue** на основе min-heap
- Методы enqueue и dequeue за O(log n)

## Использование

## Результаты

### Производительность операций
| Операция | Сложность |
|----------|-----------|
| Вставка | O(log n) |
| Извлечение корня | O(log n) |
| Построение кучи | O(n) |
| Heapsort | O(n log n) |

### Ключевые выводы
1. Кучи эффективны для реализации приоритетных очередей
2. Heapsort гарантирует O(n log n) в худшем случае
3. In-place реализация Heapsort экономит память
