# Лабораторная работа 11: Алгоритмы на строках

## Описание проекта
Исследование и реализация специализированных алгоритмов для работы со строками: префикс-функция, алгоритм Кнута-Морриса-Пратта, Z-функция и алгоритм Рабина-Карпа. Сравнительный анализ производительности и решение практических задач.

## Цели работы
- Изучить методы поиска подстрок в тексте
- Реализовать вычисление префикс-функции и Z-функции
- Освоить алгоритмы КМП и Рабина-Карпа
- Провести сравнительный анализ эффективности алгоритмов
- Решить практические задачи обработки строк

## Основные функции

### Реализованные алгоритмы
- **Префикс-функция** - вычисление массива π
- **Алгоритм КМП** - поиск подстрок с использованием префикс-функции
- **Z-функция** - вычисление массива z
- **Алгоритм Рабина-Карпа** - поиск подстрок с использованием хеширования

### Анализ производительности
- Сравнение времени выполнения алгоритмов
- Исследование в худшем и лучшем случаях
- Визуализация зависимости времени от длины текста

### Практические задачи
1. **Поиск всех вхождений паттерна** - использование различных алгоритмов
2. **Поиск периода строки** - применение префикс-функции
3. **Проверка циклического сдвига** - использование Z-функции

## Использование

### Пример работы с префикс-функцией
```python
from src.prefix_function import prefix_function

text = "ababaca"
pi = prefix_function(text)
print(pi)  # [0, 0, 1, 2, 3, 0, 1]
```

### Пример поиска подстроки алгоритмом КМП
```python
from src.kmp_search import kmp_search

text = "ababcababacab"
pattern = "ababa"
result = kmp_search(text, pattern)
print(result)  # [5]
```

## Результаты

### Производительность алгоритмов
| Алгоритм | Лучший случай | Худший случай | Память |
|----------|---------------|---------------|--------|
| Префикс-функция | O(n) | O(n) | O(n) |
| КМП | O(n+m) | O(n+m) | O(m) |
| Z-функция | O(n) | O(n) | O(n) |
| Рабина-Карпа | O(n+m) | O(nm) | O(1) |

### Ключевые выводы
1. Алгоритм КМП эффективен для поиска в текстах с повторяющимися паттернами
2. Z-функция полезна для решения задач поиска периода и циклических сдвигов
3. Алгоритм Рабина-Карпа эффективен при множественном поиске с одним хешем
4. Выбор алгоритма зависит от характеристик текста и паттерна
