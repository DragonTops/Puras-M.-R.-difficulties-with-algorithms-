# Лабораторная работа 10: Графы

## Описание проекта
Исследование и реализация основных алгоритмов работы с графами. Сравнительный анализ различных представлений графов и алгоритмов обхода.

## Цели работы
- Изучить основные понятия теории графов
- Реализовать различные представления графов
- Освоить алгоритмы обхода графов (BFS, DFS)
- Реализовать алгоритмы поиска кратчайших путей и компонент связности
- Провести сравнительный анализ эффективности представлений графов

## Основные функции

### Реализованные представления графов
- **Матрица смежности** - O(V²) памяти, быстрая проверка ребер
- **Список смежности** - O(V + E) памяти, эффективный обход соседей

### Алгоритмы обхода
- **Поиск в ширину (BFS)** - кратчайшие пути в невзвешенных графах
- **Поиск в глубину (DFS)** - рекурсивный и итеративный варианты

### Алгоритмы на графах
- Поиск компонент связности
- Топологическая сортировка
- Алгоритм Дейкстры

## Использование

### Пример работы с графами
```python
from src.graph_representation import AdjacencyMatrix

graph = AdjacencyMatrix(5)
graph.add_edge(0, 1)
graph.add_edge(1, 2)

```

### Пример обхода графа
```python
from src.graph_traversal import BFS

bfs = BFS(graph)
distances = bfs.traverse(0)
print(distances)  # [0, 1, 2, -1, -1]
```
